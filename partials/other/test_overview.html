Unit Testing New E2E Angular Framework: Protractor
  • Code-level testing
  • Used to test classes/objects/services
  • Isolated/Sandboxed
  • Mocking required (angular-mocks.js)
  • Fast
  • Uses: karma
  • Assertion framework: Jasmine / QUnit / Mocha
  • Application-level testing
  • Used for integration tests
  • Requires its own special web server
  • Can only test rendererd HTML
  • Slow
  • Uses : WebDriverJS
  • Assertion Framework : Jasmine / Cucumber / Mocha (Beta support)
