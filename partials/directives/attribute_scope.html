• Defines the scope used by the directive. Several values are available:
    • false(default) : Current scope is used.
    • true: A new scope is created for the directive. Note: Multiple directives on the same level which request a new scope will get the same.
    • {}: An isolate scope is created, which does not inherit from the parent scope. Recommended when making reusable components. Contains a set of local properties derived from the parent scope. Use this notation to link parent scope properties to the directive template properties. There are three ways to declare local properties.
